# Frontend

Interface web admin Next.js pour AutoPartage Gabon.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Zustand
- React Query

## Lancer localement

```bash
npm install
npm run dev
```

Définir `NEXT_PUBLIC_API_URL` (ex: `http://localhost:4000`) dans `.env.local`.

## Écrans initialisés

- `/login` (authentification)
- `/dashboard/users` (super admin)
- `/dashboard/vehicles`
- `/dashboard/drivers`
- `/dashboard/contracts`
- `/dashboard/payments`
